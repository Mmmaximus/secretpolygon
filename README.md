# Редактор svg-файлов

## Функционал

* Общие возможности
  * Создание svg-файла любой ширины/высоты
  * Возможность открыть и редактировать готовый svg-файл.
  * Сохранение файла в формате svg.
  * Сохранение файла в формате png.

* Инструменты
  * Курсор
